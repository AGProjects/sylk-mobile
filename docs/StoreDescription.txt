Blink Mobile
============

iOS: Open, federated SIP messaging
Android: Open-source messenger and video client. Like email — for real-time comms

Fully featured SIP client based on open standards


Description
-----------


Blink mobile uses Sylk Suite to deliver real-time communications
applications using IETF SIP protocol and WebRTC specifications.  With it you
can call to any other SIP address and you can receive calls from SIP
applications and WebRTC browsers.


Features
--------

* Audio and video calls
* Audio call recording
* Online and offline messaging
* File sharing
* Support for multiple devices 
* Portrait and landscape modes
* Location sharing
* Multiparty conferencing
* Interoperable with SIP clients
* Receive calls from the web
* Classic telephony support
* End-to-end encryption for all media
