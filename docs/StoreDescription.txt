Sylk Mobile
===========

iOS: Open, federated SIP messaging
Android: Open-source messenger and video client. Like email — for real-time comms

Fully featured SIP client based on open standards

Description
-----------

Sylk Mobile is part of Sylk Suite, a set of real-time communications
applications using IETF SIP protocol and WebRTC specifications.

Sylk Mobile can place and receive audio and video calls to and from SIP
addresses and Web browsers.  File transfers are interoperable with SIP
end-points supporting MSRP media.  Messages are encrypted end-to-end using
PGP.  Mutual location can be easily shared for meeting purposes.  Audio and
video calls can be end-to-end encrypted using zRTP.


Features
--------

* Audio and video calls
* Classic telephony support
* Call recording
* Online and offline Messaging
* Advanced file transfers
* Support for multiple devices 
* Portrait and landscape modes
* Receive calls from the web
* Location-based meetup
* Multiparty conferencing
* End-to-end encryption
